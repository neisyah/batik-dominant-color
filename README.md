# batik-dominant-color
# Sistem Penentuan Warna Dominan Kain Batik Otomatis

Penelitian ini bertujuan untuk mengekstraksi warna dominan dari citra kain batik
menggunakan ruang warna CIELAB dan algoritma K-Means Clustering.

## Metode
1. Input citra batik
2. Konversi RGB ke CIELAB
3. Klasterisasi warna menggunakan K-Means
4. Visualisasi warna dominan

## Tools
- Python
- OpenCV
- Scikit-learn
- Matplotlib
